【備忘録】 aumixのビルド

--- Internet Linked Radio Amateur ---

aumix home

最新バージョンは2.8。といっても2002年11月25日のもの。それからずっとアップデートされていないほど古いソフトウェア。

mv /usr/bin/aumix /usr/bin/aumix.old　（念のため，既存のものをバックアップ）

bunzip2 aumix-2.8.tar.bz2
tar -xvf aumix-2.8.tar
cd aumix-2.8
./configure --without-gtk --without-gtk1 --disable-nls
make
make install
ln -s /usr/local/bin/aumix /usr/bin/aumix　（make installでは，/usr/local/binにインストールされるが，IRLPでは /usr/bin/aumix の絶対パスを使っているため）

aumixは設定後quit する前に shift-S で設定を保存。quit後念のためコマンドラインから aumix -S で保存。

aumixの設定は/etc/aumixrc に保存されるが，このファイルはシンボリックリンクとして$HOMEにも存在する。

[repeater@optiplex ~]$ pwd
/home/irlp
[repeater@optiplex ~]$ ls -al .aumixrc
lrwxrwxrwx 1 root root 12 Aug 24  2008 .aumixrc -> /etc/aumixrc
[repeater@optiplex ~]$

さらに，

[root@optiplex ~]# pwd
/root
[root@optiplex ~]# ls -al .aumixrc
lrwxrwxrwx 1 root root 12  8月 24  2008 .aumixrc -> /etc/aumixrc
[root@optiplex ~]#

のように，rootのhome directoryにも同じようにシンボリックリンクが張られている。これは，irlp-install，あるいはirlp-reinstallスクリプト内で実行されている。

つまりIRLPでは，aumixを利用するユーザーがrepeaterでも，rootでも，サウンド設定を同じにするための工夫と考えられる。
