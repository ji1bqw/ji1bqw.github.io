ノードのリアルタイムステータス

--- Internet Linked Radio Amateur ---

自分のノードが今どのようなステータスになっているのかを把握できると、ホームページ上などでリアルタイムでお知らせできて良いですね。それには、$HOME/localディレクトリーの中を見るとわかります。このディレクトリには、その時々のノードの状況に応じて以下のファイルが作られたり、削除されたりします。

enable:
    ノードがエネーブルされると、このファイルが作られます。中身は空です。スクリプトから if [ -f $LOCAL/enable ] ; then でエネーブル時の処理を書くことが出来ます。ノードがディスエーブルされると消去されます。

active:
    ノードが接続されると、このファイルが作られ、ファイル中に接続相手のノード番号が記述されます。enableと同様にスクリプトから if [ -f $LOCAL/active ] ; then で、コネクト中かどうかを判断でき、さらに cat $LOCAL/active で接続相手先ノード番号を入手できます。ノードがディスコネクトされると消去されます。

calledip:
    こちらから接続した場合に、このファイルが作成され、ファイルの中身に相手のIPアドレスが記述されます。ノードがディスコネクトされると消去されます。

connectedip:
    相手から接続された場合に、このファイルが作成され、ファイルの中身に相手のIPアドレスが記述されます。ノードがディスコネクトされると消去されます。

mountコマンドを実行してみるとわかるのですが、$LOCAL ディレクトリはRAMドライブになっています。

/dev/ram0 on /home/irlp/local type ext2 (rw)

すなわち、このディレクトリは読み書きの速度は十分速いのですが、ストレージには保存されません。逆に保存する必要の無い、ノードステータスのような動的な情報がここに置かれます。
